## Cara Menjalankan Program
* Silahkan membuka cmd (Command Prompt) ke folder program dan jalankan perintah "mvn clean javafx:run"
* Tunggu beberapa saat dan program akan terbuka dengan sendirinya

## Cara Menggunakan Program
* Silahkan Tunjukkan pose tangan batu, gunting, kertas ke arah kamera
* Program secara acak akan mengenali pose tersebut dan akan melawannya (Memilih batu, gunting, kertas) secara acak
* Selanjutnya kita bisa memainkannya lagi dengan opsi "try again" atau langsung keluar dari program dengan opsi "exit"
